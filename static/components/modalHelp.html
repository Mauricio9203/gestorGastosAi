Form and Variable Descriptions
Form Buttons:
  • Submit: Use this button to submit the form after filling out all the required fields. Ensure all the information is correct and complete before clicking.
  • Fill Fields: This button automatically populates the form with default or sample values. It is particularly useful for testing or learning purposes.
Variable Descriptions:

The table below provides detailed information about each variable in the form, including its description, type, and possible values.

If you are using a mobile device, you can scroll horizontally through the table by swiping with your finger.

Variable Description Type Possible Values
age The person's age. Numeric 18-120 years
job The type of occupation (e.g., technician, admin, etc.) Categorical "technician", "admin", "services", etc.
marital The person's marital status. Categorical "single", "married", "divorced"
education The person's education level. Categorical "primary", "secondary", "tertiary"
default History of payment defaults. Categorical "yes", "no"
balance The person's bank balance. Numeric Numeric value
housing Whether the person has a mortgage. Categorical "yes", "no"
loan Whether the person has a personal loan. Categorical "yes", "no"
contact The type of contact used for reaching the person. Categorical "cellular", "telephone"
day The day of the month the last contact occurred. Numeric 1-31
month The month of the last contact. Categorical "jan", "feb", "mar", etc.
duration Duration of the call in seconds. Numeric Numeric value
campaign Number of contacts made during the campaign. Numeric Numeric value
pdays Days since the last contact. Numeric -1 means no previous contact
previous Number of previous contacts before this campaign. Numeric Numeric value
poutcome Outcome of the previous campaign. Categorical "success", "failure", "nonexistent"
deposit Target variable: whether a person subscribed to a term deposit. Categorical "yes", "no"
